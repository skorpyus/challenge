# Challenge Test
 Technical Challenge Test - Aplicacion

## Scripts

Se pueden ejecutar los siguientes scripts

### `npm start`

Para levantar la api en development mode.\
Se abre en el browser [http://localhost:8080] 

### `npm build`

Construye el entregable distribuible

## Requerimiento

Un App cliente en React que permita enviar textos a la API del proyecto adjunto en el mismo repo y muestre las respuestas realizadas a la API desde la ultima a la primera.
